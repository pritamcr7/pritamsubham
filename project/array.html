Arrays

Definition:

It is a variable which contains a set of same type of elements and all elements are stored in a sequential memory location. Elements are referenced by a set of sequential number called index.

Syntax:

type    nameofarrayvariable[Size];            eg: int arr[5];

type=valid type

nameofarrayvariable=variable name

Size=It may contain either a positive integer constant/macro template equivalent to positive integer constant.

Initialization:

int arr[5]={22,25,30,12,18};
or
char arr[5]={'b','c','d','g','a'};

Accessing Array Element:

int age[5];
int n = 0;
int m=5;//m=size of the array
for(n=0;n=m-1;n++)
{
age[n] = n;
}
