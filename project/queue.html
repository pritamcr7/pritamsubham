QUEUE

Definition:

Queue is a linear data structure in which insertion and deletion takes place from two different ends.One end used for insertion and is called Rear end and the other end is used for deletion and is called Front end.

Operation:

There are 2 operations associated with Stack.

          1.Insertion:Insert the element in the Rear end

          2.Deletion:Delete the element from Front end.













Queue is known as First In First Out(FIFO) format because the First inserted item in queue is deleted first

Queue Representation:

There are 2 ways to represent a Queue.
          1.Using Array:Insertion and Deletion performed using array.
          2.Using Linked List:Insertion and Deletion performed using linked list.

Queue as an Abstract Data Type(ADT):

Queue is known as ADT bescause in Queue we always focus on some operation in Queue like Insertion and Deletion but ignore the how the operatios are designed.
