Stack

Definition:

Stack is a linear data structure in which insertion and deletion takes place at one end.This end is called Top end .

Operation:

There are 2 operations associated with Stack.

          1.Push:Insert the element in the Top end

          2.Pop:Delete the element from Top end.


























Stack is known as Last In First Out(LIFO) format because the Last inserted item in stack is deleted first

Stack Representation:

There are 2 ways to represent a Stack.
          1.Using Array:Push and Pop performed using array.
          2.Using Linked List:Push and Pop performed using linked list.

Stack as an Abstract Data Type(ADT):

Stack is known as ADT bescause in stack we always focus on some operation in stack like push and pop but ignore the how the operatios are designed

Uses of Stack:

Stack is mostly used for converting and evaluting expression
          1.Infix.
          2.Postfix
          2.Prefix
